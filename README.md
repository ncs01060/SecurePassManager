# 비밀번호 관리 프로그램 매뉴얼

본 프로그램은 개인의 암호화된 비밀번호를 안전하게 저장하고 관리할 수 있도록 설계된 로컬 데스크톱 프로그램입니다.

## 주요 기능
1. **마스터 비밀번호 로그인**: AES 256비트 Fernet 암호화 알고리즘을 사용해 모든 데이터가 마스터 비밀번호로 암호화됩니다. 이 마스터 암호를 잊으면 암호를 복구할 수 없습니다.
2. **비밀번호 검색 및 보안 복사**: 저장된 이메일 또는 사이트를 검색하여 더블 클릭하는 순간 복호화되어 임시 클립보드에 비밀번호가 안전하게 복사됩니다.
3. **안전한 비밀번호 추천(생성)**: 새로운 사이트 등록 시 "Generate Safe" 버튼을 누르시면, 기존 비밀번호와 절대 겹치지 않는 16자리의 강력한 랜덤 암호를 추천받을 수 있습니다.
4. **접속한 사이트 암호 자동 알림 (복사 감지)**: 브라우저 등에서 URL(`https://...`)을 복사하거나 주소를 복사하면, 해당 사이트가 등록되어 있을 경우 **자동으로 알림이 발생하고 비밀번호가 클립보드에 클립됩니다**. 바로 붙여넣기(Cmd+V)하여 로그인할 수 있습니다.

## 실행 방법
### 1. 가상환경 활성화 (최초 실행 또는 터미널 재실행 시)
Mac 터미널을 열고 다음 명령어를 입력해주세요:
```bash
cd 사용하고자 하는 폴더명
source venv/bin/activate
```

### 2. 프로그램 실행
가상환경 활성화 상태에서 다음 명령어를 입력합니다:
```bash
python3 app.py
```

### 3. 사용 방법 요약
- **최초 실행 시**: 설정할 `Master Password`(마스터 비밀번호)를 입력하게 됩니다. 이를 절대 잊어버리지 마세요!
- **사이트 추가**: `Add New Password` 버튼을 누르고 `Site`, `Username`, `Password`를 입력 후 저장하세요.
- **클립보드 스니핑 (추가 기능)**: 사파리나 크롬 브라우저 상단 주소창을 복사해보세요(Cmd+C). 앱이 백그라운드에 켜져 있다면, 즉시 알림창(Notification)으로 감지 여부를 알려주며 바로 붙여넣기가 가능하게 만들어 줍니다.
