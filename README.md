# 🛡️ SecurePassManager (안전한 비밀번호 관리자) 사용 매뉴얼

**SecurePassManager**는 강력한 암호화 기술과 실시간 브라우저 연동 기능을 결합한 스마트 비밀번호 관리 시스템입니다. 복잡한 파이썬 환경 설정 없이, 컴파일된 실행 파일(`.exe` 또는 `.app`)만으로 즉시 사용할 수 있도록 설계되었습니다.

---

## 🚀 프로그램 실행 방법

별도의 설치 과정이 필요 없는 포터블(Portable) 프로그램입니다. 사용하시는 운영체제에 맞는 실행 파일을 클릭만 하면 바로 켜집니다.

- **Windows 사용자:** 제공된 `SecurePassManager.exe` 파일을 더블 클릭하여 실행합니다.
- **macOS 사용자:** 제공된 `SecurePassManager.app` 아이콘을 클릭하여 실행합니다. (또는 `SecurePassManager.command` 실행)

> **주의사항:** 프로그램 구동 시 최초 1회, 사용자의 소중한 데이터를 담을 로컬 데이터베이스 파일이 같은 폴더 내에 안전하게 자동 생성됩니다.

---

## ✨ 핵심 기능 안내

1. **🔒 완벽한 암호화 보호:** 최초 설정된 '마스터 비밀번호'를 기반으로 모든 데이터를 뚫을 수 없는 수준인 AES-128 방식으로 암호화 구축을 하였습니다. (단, 마스터 비밀번호를 분실하면 복구가 불가능하니 주의하세요.)
2. **🌐 스마트 브라우저 연동:** 사용자가 웹브라우저(Chrome 등)에서 방문 중인 웹사이트를 인식하고, 앱에 저장된 사이트라면 "비밀번호를 복사할까요?"라고 자동 알림을 띄웁니다.
3. **📋 클립보드 감지 복사:** 주소창의 URL을 `Ctrl+C` 또는 `Cmd+C`로 복사하기만 해도, 백그라운드에서 이를 감지하여 해당 사이트의 비밀번호를 클립보드에 자동으로 올려줍니다.
4. **🎲 안전한 난수 암호 생성:** 해커가 유추하기 힘든 조합의 무작위 비밀번호를 클릭 한 번으로 손쉽게 생성할 수 있습니다.
5. **🎨 파비콘(Favicon) 썸네일 지원:** 사용자 편의성을 위해 등록한 웹사이트의 공식 로고 이미지를 자동으로 가져와 목록에 예쁘게 띄워줍니다.
6. **👻 백그라운드 구동:** 창의 "X"를 눌러 닫더라도 프로그램이 완전히 꺼지지 않고 시스템 트레이(알림 영역)에 상주하여 클립보드 감지와 같은 기능을 계속 제공합니다.

## 📖 단계별 사용 가이드 (User Guide)

### 단계 1. 마스터 비밀번호 설정 (최초 실행 시)
1. 프로그램을 처음 실행하면 **"마스터 비밀번호 설정"** 창이 나타납니다.
2. 앞으로 저장할 모든 사이트 정보와 비밀번호를 보호할 '단 하나의 마스터 키' 이므로 **절대 잊어버리지 않는 강력한 비밀번호**를 입력하고 확인을 누릅니다.

### 단계 2. 웹사이트 계정 및 비밀번호 추가하기
1. 상단의 **[새 비밀번호 추가]** 버튼을 클릭합니다.
2. `URL(사이트 주소)` 칸에 로그인할 사이트의 도메인(예: `naver.com`, `github.com`)을 정확히 입력합니다.
3. 사용하는 `사용자 ID`와 `비밀번호`를 기입합니다.
    - *Tip: 보안이 걱정되신다면, 하단의 **[안전한 암호 생성]** 버튼을 눌러 강력한 패스워드를 자동으로 부여받으세요.*
4. **[저장]**을 누르면 리스트에 자동으로 아이콘이 붙어서 등록됩니다.

### 단계 3. 일상 생활에서 스마트하게 활용하기
1. 비밀번호를 등록해둔 상태에서 브라우저를 켜서 서핑을 합니다.
2. SecurePassManager에 등록해둔 사이트의 로그인 화면에 진입하거나 주소를 복사하면,
3. **앱이 화면 귀퉁이에 "비밀번호를 복사할까요?" 라고 팝업**을 띄웁니다.
4. "예"를 누르고 해당 웹사이트의 비밀번호 칸에서 `Ctrl+V`(붙여넣기)를 하면 곧바로 완료됩니다.
5. 앱 내부의 비밀번호 리스트에서 특정 항목을 더블클릭해도 비밀번호가 클립보드에 바로 복사됩니다.

### 단계 4. 프로그램 완전히 종료하기
메인 화면의 우상단 `X`를 누르더라도, 빠른 사용과 감지 기능을 돕기 위해 프로그램은 알림 영역에 계속 실행되어 있습니다. 완전히 종료하고 싶다면:
- **Windows:** 화면 우측 하단 시계 옆의 시스템 트레이에서 자물쇠 모양 아이콘을 우클릭 ➔ **"종료"** 메뉴 클릭
- **macOS:** 화면 우측 상단 메뉴 막대의 자물쇠 모양 아이콘 좌클릭 ➔ **"종료"** 메뉴 클릭

## 🛠 (개발자용) 파이썬 빌드 가이드

본 지침은 일반 사용자가 아닌, 소스 코드를 통해 직접 `.exe` 및 `.app` 파일을 도출하고자 하는 학생/개발자를 위한 가이드입니다.

1. 의존성 설치: `pip install -r requirements.txt`
2. 개발 테스트: `python app.py`
3. 프로그램 빌드(패키징):
   - Windows 환경은 `build_windows.bat`를 실행하여 `dist/` 내에 `.exe` 생성
   - macOS 환경은 `build_mac.sh`를 실행하여 `.app` 파일 추출
