# 🛡️ SecurePassManager (안전한 비밀번호 관리자)

**SecurePassManager**는 강력한 암호화 기술과 실시간 브라우저 연동 기능을 결합한 스마트 비밀번호 관리 시스템입니다. macOS 환경에 최적화되어 있으며, 사용자가 웹사이트에 접속할 때 자동으로 비밀번호를 준비해줍니다.

---

## ✨ 주요 기능 (Key Features)

### 1. 🌐 실시간 Chrome 브라우저 감지 (macOS 특화)
- **자동 사이트 인식:** Chrome 브라우저에서 현재 보고 있는 사이트를 실시간으로 감지합니다.
- **스마트 팝업:** 저장된 비밀번호가 있는 사이트에 접속하면 "비밀번호를 복사할까요?"라는 확인 팝업을 띄워줍니다.
- **시각적 알림:** 상단 상태바가 색상(초록/노랑)으로 실시간 저장 여부를 알려줍니다.

### 2. 🎨 스마트 파비콘(Favicon) 시스템
- **아이콘 자동 매칭:** 사이트 도메인을 분석해 해당 사이트의 공식 아이콘을 목록에 표시합니다.
- **로컬 캐싱:** 한 번 가져온 아이콘은 `icons/` 폴더에 저장되어 오프라인에서도 빠르게 로드됩니다.

### 3. 🔒 강력한 보안 및 암호화
- **마스터 비밀번호:** 앱 시작 시 설정한 마스터 비밀번호 없이는 어떤 데이터도 복호화할 수 없습니다.
- **AES 암호화:** 모든 유저 비밀번호는 `cryptography` 라이브러리를 통해 강력하게 암호화되어 SQLite DB에 저장됩니다.

### 4. 📋 지능형 클립보드 및 트레이 지원
- **URL 감지:** 브라우저 주소를 복사하면 해당 사이트의 비밀번호를 즉시 클립보드에 복사해줍니다.
- **백그라운드 실행:** 창을 닫아도 시스템 트레이(Tray)에서 계속 동작하며 알림을 제공합니다.
- **검색 및 관리:** 사이트명으로 간편하게 검색하고, 더블 클릭만으로 비밀번호를 복사할 수 있습니다.

## 🚀 시작하기

### 1. 필수 라이브러리 설치
```bash
pip install -r requirements.txt
```

### 2. 프로그램 실행
```bash
python3 app.py
```

## 📖 사용 매뉴얼 (User Guide)

1.  **최초 설정:** 
    - 처음 실행 시 사용할 '마스터 비밀번호'를 설정합니다. 이 비밀번호는 반드시 기억해야 합니다.
2.  **암호 추가:** 
    - `새 비밀번호 추가` 버튼을 눌러 사이트 주소(도메인), 아이디, 비밀번호를 입력합니다.
    - `안전한 암호 생성` 버튼을 통해 강력한 랜덤 비밀번호를 만들 수 있습니다.
3.  **자동 복사 활용:** 
    - Chrome 브라우저를 띄워놓고 저장된 사이트로 이동해 보세요. 앱이 자동으로 감지하여 복사 여부를 묻습니다.
4.  **권한 설정 (macOS):**
    - 실시간 감지 기능을 위해 앱이 Chrome을 제어할 수 있도록 시스템 권한 요청 시 `확인`을 눌러주세요.
5.  **트레이 이용:**
    - 메인 창을 닫아도 메뉴 막대의 아이콘을 통해 프로그램을 계속 활성화해 둘 수 있습니다.

## 🛠 기술 스택
- **Language:** Python 3
- **UI Framework:** PyQt6
- **Database:** SQLite3
- **Encryption:** PBKDF2 (Key Derivation), Fernet (AES-128)
- **Dependencies:** cryptography, pyperclip, PyQt6
